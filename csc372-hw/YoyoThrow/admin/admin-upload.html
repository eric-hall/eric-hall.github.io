Bulk Upload Products

To bulk upload products, please select a JSON file that contains an array of product objects. All fields are required, except for the url field, which can be null. Each product must have a unique id. The JSON format should look like this:


    [
        {
            "id": 1,
            "name": "AceYo Mercy 4",
            "description": "Competition-ready.",
            "category": "yo-yo",
            "image": "resources/images/yoyo1.png",
            "price": 59.99,
            "url": "details.html"
        }
    ]
